FastAI Webinar Series - Deep Learning Tutorials

Saturday, December 8, 2018

Details:

FastAI is a deep learning library that has gained a lot of popularity in recent months, because of it's easy to use high level API and great performance. It is built on top of PyTorch. This webinar series will consist of walkthroughs of the Jupyter notebooks used for development of library, building things bottom-up from the scratch.

Recordings will be made available here

Time:

11:00AM-12:30 PM IST, every Saturday

Speaker:

Aakash N S

Aakash is the co-founder and CEO of SwiftAce, a computer vision startup. Prior to starting SwiftAce he worked as software engineer at Twitter in Ireland & San Francisco. He graduated from IIT Bombay in 2013. He’s also an avid blogger and open source contributor.

Prerequisites:

* Basic knowledge of Python

* Basic knowledge of Machine Learning
